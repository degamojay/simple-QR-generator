This repository is a Simple URL QR generator based on the URL provided by the user. For this to run, ensure that nodejs is installed. Then, 
1. run node index.js in your terminal.
2. enter the URL that you want to be generated as a QR code.
3. Voilah! a filename qr-img.png is now generated according to user's request.
4. Scan it to test if it works.
   
Thanks for checking it out!
